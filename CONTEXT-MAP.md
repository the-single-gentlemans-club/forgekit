# Context Map

ForgeKit is a multi-context monorepo. Each context has its own `CONTEXT.md` file:

| Context | Location | Purpose |
|---------|----------|---------|
| async | `packages/async/CONTEXT.md` | Async utilities |
| colors | `packages/colors/CONTEXT.md` | Color manipulation |
| strings | `packages/strings/CONTEXT.md` | String utilities |
| utils | `packages/utils/CONTEXT.md` | General utilities |

Shared architecture decisions live in `docs/adr/`.

Read the appropriate `CONTEXT.md` when working on a package, or check `docs/adr/` for system-wide decisions.
